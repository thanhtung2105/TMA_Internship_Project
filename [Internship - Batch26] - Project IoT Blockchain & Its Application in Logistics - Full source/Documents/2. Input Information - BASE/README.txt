Phần nhập chủ yếu được chạy thông qua một GUI Java (đã có source code sẵn và API).
Java GUI này được quản lý và đóng gói bằng Gradle, nên để tiếp tục phát triển trên các nền 
tảng như Mobile hoặc các máy tính khác, cần xem Project GUI với API mẫu để sử dụng và 
Dev được trên các nền tảng khác phù hợp. 

Trong việc đóng gói GUI bằng Gradle sẽ có phần liên kết trực tiếp giữa Java và Ethereum Blockchain.
Có import tài khoản và import API của Infura để làm máy chủ tạo block lên Rinkeby TestNet.
